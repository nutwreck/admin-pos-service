# Golang POS

API backend rest Admin POS Service using Gin Framework and Gorm ORM Database.

## Command

- ### App Lifecyle

```sh
$ go run main.go || make go run
```

- ### Docker Lifecycle

```sh
$ docker-compose up -d --build
```

- ### Swagger

```sh
$ go get -u github.com/swaggo/swag/cmd/swag
$ $HOME/go/bin/swag init
```
